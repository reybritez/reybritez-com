Estas Boludeces aims to enhance the checkout process for an online commerce store by integrating WhatsApp as a communication channel. The primary objectives include streamlining the purchasing experience for customers and providing them with the convenience of communicating directly through WhatsApp.

** How to use this project code **

1. **Clone the repository:**

   - Open your terminal or command line.
   - Use the `git clone` command followed by the repository URL to clone it to your local machine.

   ```bash
   git clone https://github.com/reybritez/estasboludeces.git
   ```

2. **Create a copy of this spreadsheet:**

   - Access the provided spreadsheet.
   - Make a copy of this spreadsheet in your own workspace.

3. **Publish the spreadsheet to the web as a CSV file:**

   - In the spreadsheet, go to "File" on the menu bar.
   - Select "Publish to the web" and choose "Comma-separated values (.csv)" from the dropdown menu.
   - Click "Publish" and make sure the generated link corresponds to a CSV file.

4. **Copy the CSV file link:**

   - Once published, ensure that the generated link is of type "Comma-separated values (.csv)".
   - Copy this link as you'll need it later.

5. **Fill out the file and complete your store's data:**

   - Download the provided `.env.example` file from the repository.
   - Fill in the required data with your store's information.
   - Rename this file to `.env.local`.

6. **Buy a domain on sites like Namecheap:**

   - Visit a domain registrar website, such as Namecheap.
   - Search for and purchase the domain you desire for your project.

7. **Publish the site on a hosting service that supports Next.js, like Vercel:**
   - Go to the Vercel website and create an account if you don't already have one.
   - Follow the steps to upload your Next.js project to Vercel.
   - During the deployment process, ensure you configure the environment variables correctly using the `.env.local` file.
   - Once deployed, your project will be online and accessible through the domain you purchased.

These steps should guide you through the process of using the project, from cloning it to putting it online. Good luck with your project!

This project combines an efficient checkout process with WhatsApp integration to provide customers with a seamless and personalized shopping experience. By leveraging the power of real-time communication and analytics, the project aims to drive sales, improve customer satisfaction, and foster long-term relationships with customers.
